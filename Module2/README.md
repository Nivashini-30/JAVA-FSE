What is Big O Notation?
      Big O notation is used to measure the efficiency of an algorithm by describing how its execution time grows as the input size increases. It helps compare algorithms and predict performance for large datasets.

Which Algorithm is best?
      Binary Search is more suitable for an e-commerce platform because it provides faster search performance for large datasets. However, the product data must be sorted before applying binary search. Linear search is simple but becomes inefficient as the number of products increases.

Comparision of Best,Average,Worst Cases:
Linear Search:
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)
Binary Search:
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)
