Exercise 1: Logging Error Messages and Warning Levels

Objective

To demonstrate logging of error and warning messages using the SLF4J logging framework with Logback implementation.

Technologies Used

Java
SLF4J API 1.7.30
Logback Classic 1.2.3
Logback Core 1.2.3

Description

This application demonstrates how to use SLF4J for logging messages at different severity levels. The program logs an error message and a warning message using a Logger object created through LoggerFactory.

Files

LoggingExample.java

Logging Levels Used

ERROR: Used to log critical issues and application errors.
WARN: Used to log warning messages that may require attention.

Expected Output

ERROR LoggingExample - This is an error message

WARN LoggingExample - This is a warning message

Conclusion

Successfully implemented logging functionality using SLF4J and Logback to record error and warning messages in a Java application.
