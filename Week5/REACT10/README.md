# Office Space Rental App (React Hands-On 10)

A React application built to demonstrate core React concepts, specifically utilizing JSX syntax to create elements, map attributes, and render the DOM dynamically[cite: 4]. 

## Features & Objectives

* **JSX Elements & Attributes:** Displays page headings and renders image elements using JSX attributes (`src`, `width`, `height`)[cite: 4].
* **Dynamic Inline CSS & Class Binding:** Evaluates rental prices dynamically. Text color automatically changes to Red if the rent is below or equal to 60,000, and Green if it is above 60,000[cite: 4].
* **JavaScript Expressions in JSX:** Creates an object for the office space (Name, Rent, Address) and uses JavaScript expressions `{}` to render the details seamlessly into the DOM[cite: 4].
* **Array Mapping:** Loops through a list of office space objects to display multiple properties efficiently[cite: 4].
