# Student Management Portal (StudentApp)

This repository contains a multi-component React application built as part of the Component Development hands-on lab[cite: 2].

## Objective
The primary objective of this task is to create a multi-component workspace layout, implementing class components (`Home`, `About`, and `Contact`) and calling them collectively inside the primary `App` root layout component layout[cite: 2].
