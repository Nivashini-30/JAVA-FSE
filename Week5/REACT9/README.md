# Cricket App (ES6 Features & Destructuring)

A React application designed to demonstrate essential ES6 features including arrays `.map()`, arrow functions, destructuring variables, and array merging using the spread operator[cite: 3].

## Features & Objectives

* **ES6 Map & Arrow Functions:** Outputs player profiles and filters subset structures (such as players with scores below 70) dynamically[cite: 3].
* **Destructuring Properties:** Destructures position arrays cleanly to separate players into odd-indexed and even-indexed elements[cite: 3].
* **Array Merging:** Uses the ES6 spread operator (`...`) to merge two distinct arrays (T20 and Ranji players) into a single display lineup[cite: 3].
