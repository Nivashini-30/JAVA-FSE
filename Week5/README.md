# My First React Application (myfirstreact)

This repository contains a basic Single-Page Application (SPA) built as part of the introductory React hands-on lab.

## Objective
The primary objective of this task is to set up a clean React environment and render a heading that prints: 
**"Welcome the first session of React"**.

---

## Prerequisites
To run this project locally, ensure you have the following installed:
* **Node.js**
* **NPM**[cite: 2]
* **Visual Studio Code**[cite: 2]
