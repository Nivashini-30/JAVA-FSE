# GitDemo - Version Control Lab Practice

This repository contains basic version control exercises using Git Bash and Notepad++ on Windows.

## Project Overview

* Initialized a local Git repository (`GitDemo`).
* Configured Notepad++ as the default global text editor for Git commits.
* Created, staged, and committed initial workspace files (`welcome.txt`).
* Configured remote branch tracking with GitHub.
  
##  Tools Used

* **OS:** Windows
* **Terminal:** Git Bash
* **Editor:** Notepad++
* **Version Control:** Git
* 
## Commands Executed

### 1. Set Notepad++ as Default Git Editor

git config --global core.editor "\"C:/Program Files/Notepad++/notepad++.exe\" -multiInst -nosession"
